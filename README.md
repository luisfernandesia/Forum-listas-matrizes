# Jogo da Velha em Python (usando Matrizes)

Este projeto foi desenvolvido como atividade da disciplina de Programação, onde o objetivo era criar uma aplicação utilizando listas, matrizes ou ambas, para resolver um problema simples.  
A solução escolhida foi a implementação de um jogo da velha usando uma matriz 3x3 como tabuleiro.

# Sobre o Jogo

- O tabuleiro é uma matriz 3x3, composta por listas dentro de listas.
- Dois jogadores alternam jogadas:  
  - X 
  - O
- O programa verifica automaticamente:
  - Jogadas inválidas
  - Vitória por linha
  - Vitória por coluna
  - Vitória por diagonais
  - Empate
